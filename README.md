### Installation
To run the code, you will need to install the following libraries:

- pandas

- sklearn

- seaborn

- matplotlib

- numpy

You can install the libraries using the following command:

``` pip install pandas```

``` pip install scikit-learn```

``` pip install seaborn```

``` pip install matplotlib```
